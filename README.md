# 📖 The Boys seasons <a name="about-project"></a>

**The Boys Seasons** is a single yet fully functional web app developed using React for the UI. It allows the users to retrieve data from two different APIs (one for the tv show info and one for the likes and comments).

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

> You need the following tools be installed in your computer:

> - [Git](https://www.linode.com/docs/guides/how-to-install-git-on-linux-mac-and-windows/)
> - IDE

### Setup

Clone this repository to your desired folder:

```sh
  cd my-prefered-folder

  git clone https://github.com/luis-pomare/the-boys-seasons.git

```

### Install

Install this project with:

```sh
  cd blog-app

  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  npm run test
```
